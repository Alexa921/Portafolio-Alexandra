Energym – Página Web y Tienda Online de Gimnasio

Descripción:

Energym es una plataforma web diseñada para un gimnasio moderno que combina
información sobre sus servicios, entrenadores y clases, junto con una tienda online
donde los usuarios pueden adquirir productos deportivos. El sistema incluye un
carrito de compras totalmente funcional que permite agregar, eliminar y actualizar
productos en tiempo real, mostrando los cambios de precio de forma automática.

Tecnologías utilizadas:

  • HTML
  • CSS
  • JavaScript

Mi función:

  • Desarrollo completo del proyecto de manera individual, implementando tanto
    la parte informativa del gimnasio como la funcionalidad de la tienda online.
  • Programación del carrito de compras con actualización dinámica de precios
    y manejo de productos seleccionados por el usuario.
  • Diseño y maquetación con enfoque visual energético, moderno y responsive
    para ofrecer una experiencia de usuario atractiva en cualquier dispositivo.
